### 1. Generics ‚Äî What does <T> mean?

Generics mean type safety.
They tell Dart what type of data you are working with.

## Example:
```sql
List<String> names = ["Samir", "Rahim", "Karim"];
```

Here:
String is the generic type
This list can only store strings

‚ùå Wrong:
```sql
names.add(10); // Error
```
---

### 2. Lists ‚Äî Storing Multiple Values

A List stores multiple items in one variable.

## Example:
```sql
List<int> numbers = [1, 2, 3, 4];
```

Common operations:

```sql
numbers.add(5);        // Add item
numbers.remove(2);     // Remove item
print(numbers.length); // Total items
```

---

### 3. Using Generics with Widgets

## Example:

```sql
List<Widget> items = [
  Text("One"),
  Icon(Icons.star),
  ElevatedButton(onPressed: () {}, child: Text("Click")),
];
```

This list can only contain widgets.

### 4. Adding Gradient Colors in Flutter üé®

Gradients are used to create color transitions.

## Example (Linear Gradient):

```dart
Container(
  height: 200,
  width: double.infinity,
  decoration: BoxDecoration(
    gradient: LinearGradient(
      colors: [Colors.blue, Colors.purple],
      begin: Alignment.topLeft,
      end: Alignment.bottomRight,
    ),
  ),
)
```

---

### 5. Combining Lists + Generics + Gradients

## Example:

```dart
List<Color> gradientColors = [Colors.red, Colors.orange];

Container(
  decoration: BoxDecoration(
    gradient: LinearGradient(colors: gradientColors),
  ),
)
```
